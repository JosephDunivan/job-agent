# Job Agent

A career intelligence agent that goes beyond "click apply" — combining OSINT-powered networking, AI resume tailoring, skill gap analysis, and pipeline management into an autonomous job search system.

## What This Is

Most job automation tools are apply bots that spam applications. Job Agent is a **career intelligence system** that:

- **Maps the landscape** — uses OSINT to understand who's at target companies, find warm paths, and discover events
- **Builds relationships** — recommends outreach, tracks interactions, identifies connections
- **Develops the candidate** — identifies skill gaps, recommends projects, builds portfolio
- **Manages the pipeline** — from discovery through negotiation, not just "click apply"
- **Learns over time** — tracks what works and adapts

## Status

**Early scoping phase.** We're refining requirements and defining architecture before writing code. See [PROJECT_NOTES.md](./PROJECT_NOTES.md) for the full scoping document with research, decisions, and progress.

## Principles

- Open source first, low cost
- Self-hosted where possible (Ollama for LLM, n8n for orchestration, SpiderFoot for OSINT)
- Privacy-aware — your career data stays on your machine
- Phased delivery — useful at each stage, not all-or-nothing
